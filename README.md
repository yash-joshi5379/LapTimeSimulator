This repository contains all files required to run simulations, animate car trajectories, visualise plots, and analyse metrics by comparing to IndyCar data.

# Project Structure
| File                     | Purpose                                                                                                            |
| -------------------------| ------------------------------------------------------------------------------------------------------------------ |
| **B1965run52**           | Contains real tyre data.                                                                                           |
| **Car**                  | Contains all attributes and methods related to the vehicle.                                                        |
| **multiple_simulations** | Runs multiple simulations and analyses simulation metrics.                                                         |
| **one_simulation**       | Runs one simulation and creates plots and animates car trajectory.                                                 |
| **Simulator**            | Contains all attributes and methods related to the simulation.                                                     |
| **Track**                | Contains all attributes and methods related to the track.                                                          |

# Simulation Instructions

To run one complete simulation, simply open the file named `one_simulation.m` and press the green play button to start the process.

To run multiple simulations (Monte Carlo method), simply open the file named `multiple_simulations.m` and press the green play button to start the process.
